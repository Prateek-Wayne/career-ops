### Sell senior without lying
- Lead with Philips backend impact: Spring Boot services across 7+ microservices, API latency improvement, and production incident ownership.

- If the role expects mentoring, PoCs, and broad SDLC ownership, ask for clarity on promotion path and a 6-12 month review plan.
- Accept only if compensation is fair for a high-ownership SDE-II in Pune and the backend scope is real.
